# java-file-handling
Java File Handling using FileWriter and BufferedReader
This Project Demonstrates basic file handling in java.

## Features
 - Create a text file
 - Write data using Filewriter
 - Read data using FileReader & BufferedReader
 - Display output in console
 - Handle Exceptions Properly

## Tools used
 - Java
 - Eclipse IDE
